Advice for 10-20 year olds

Patrick Collision

Try to get some sense of kinds of things you enjoy doing

If you're 10–20: these are prime years, go deep on things, become an expert. In particular, try to go deep on multiple things. One of the main things you should try to achieve by age 20 is some sense for which kinds of things you enjoy doing.more →
